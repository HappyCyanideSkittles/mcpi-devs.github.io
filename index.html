Frequently Asked Questions

What is Minecraft Pi?

It's an fork (modification) of an old versio of Minecraft Pocket Edition (v0.6.1), ported to the Raspberry Pi.


Can I play it?

It depends on your platform: If you are in a Raspberry Pi 2 or superior, with Raspberry Pi OS (previously Raspbian) installed as one of your operating systems, yes! Otherwise, you can use [MCPI-Docker](), which, as its name says, requires Docker, or you can use the MCPI-Docker mods in any other RPi emulator (QEMU, QEMU + KVM).


How do I get it?

If you have an updated RPi OS Full version, you already have it! If you have an updated RPi OS _without_ MCPI preinstalled, you can install it by running the following command:

sudo apt-get install minecraft-pi

Otherwise, if you are using MCPI-Docker, it will install MCPI automatically.


What extra features does MCPI have?

Minecraft Pi comes with a built-in modding API, with clients in multiple languages: Python ([mcpi](https://github.com/martinohanlon/mcpi), [Picraft](https://picraft.readthedocs.io/)), [JavaScript](https://github.com/mdnorman/node-mcpi), Java and probably more.

How is it related to MCPE?

Minecraft Pi is related to Minecraft Pocket Edition, because the first is a "modified" and ported version of the second one, but they are **not** the same. For example, MCPI by default hasn't the Survival features enabled.


And what is this?

This is the main webpage of MCPI-Devs, a community dedicated to Minecraft Pi (and eventually PE) modding. If you are interested, come join us! In Discord, Reddit and our forums. We also appreciate contributions and feedback on our projects in GitHub!
